# GX-builder
### This is the tool you need to make your life easier while you build your own custom Opera GX mod!
## Requirements:
### Install [Python](https://www.python.org/downloads/)
## Guide:
#### 1- Download the latest release from the [Releases](https://github.com/Open-GX/GX-builder/releases) (Pre releases could contain bugs or untested feutures, please be ware that we will solve the issues, everything else works!)

#### 2- Open (or execute) the builder.py
#### 3- Please prepare your wanted files (like the mp4 for your splash or wallpaper, the images and etc)

#### 4- Put your developer name, mod name, version and author name in the info box!
![Builder](BUILDERGUIDEPNG/one.png)

#### 5- Put your desired name in the box below the ID when you want to add your files (there are boxes with the options after the import section below the white screen)
![Builder](BUILDERGUIDEPNG/two.png)
![Builder](BUILDERGUIDEPNG/eight.png)

#### 6- Please select the "add music set" or add pack, put a name, click it, then add tracks and then make sure you select the popup window that appears and then click "OK" (Otherwise it doesn't recognise the command!)
![Builder](BUILDERGUIDEPNG/three.png)
![Builder](BUILDERGUIDEPNG/four.png)
![Builder](BUILDERGUIDEPNG/five.png)
#### Only for keyboard!!!!!-> ![Builder](BUILDERGUIDEPNG/six.png)

#### 7- Please after you've added your files go to the section Validator/Export and validate the files, check what seems to be missing if errors appear and follow them to make sure the validation is compleated (in the program).
![Builder](BUILDERGUIDEPNG/seven.png)

#### 8- Make sure that you Export(Load unpacked) your files into a folder! (REQUIRED)
#### 9- Follow the rest of the guide in how to import your mods into OperaGX in the [KittyOperaGXMOD](https://github.com/Open-GX/KittyWindowsXP-OperaGX-mod)
#### 10- Export Zip is for creators only and it doesn't work with unofficial members of OperaGX!

## File Requierments!
### 1- Make sure that the splash screen is in format mp4 with 1080x1080 resolution, to be 5 seconds, to be h.264 and to be with aac audio!
### 2- Make sure that the file 512.png is in a format 512x512!
### 3- Make sure that the keyboard sounds are .wav or .mp3 (the music and sounds are in only .mp3 format!)
